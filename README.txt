SIGPIM-SLZ MVP Refinado (HTML/CSS/JS puro)

Como usar:
1. Abra o arquivo index.html no navegador.
2. Entre com qualquer e-mail demo mostrado na tela inicial.
3. Para restaurar o estado original da apresentação, clique em "Restaurar demo".

O que foi refinado:
- tela inicial de login/apresentação
- dashboard melhorado
- filtros em imóveis
- ficha de imóvel com abas
- edição rápida de imóvel
- gestão básica de usuários
- ações demo para documentos e pendências
- impressão fácil para relatórios/demonstração
